# PersonalNameServer
A simple, lightweight web application to save, map, and quickly access URLs using custom names.
Think of it like a personal DNS for URLs — shield long, complicated links behind easy-to-remember names.

## v0.1 – First Release

### Features
* Save URLs with custom names for easy access.
* Search for saved URLs instantly.
* Responsive web interface for browsers.
* Optionally use CLI version to add URLs.
* Dockerized for easy deployment anywhere.

### Why Use PNS?
* Protects users from long, ugly URLs.
* Makes URL management personal and fast.
* Lightweight and easy to host locally or for small teams.
* No complex setup — just Docker .
  
### Prerequisites
* Docker installed
* Optional: Docker Compose for multi-container setup
* 1 GB RAM recommended, 2 CPUs preferred for smooth testing

### Running on Your Local System
#### run:
* git clone https://github.com/Mosh-py/PersonalNameServer.git
* cd PersonalNameServer
